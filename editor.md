

**Shortcuts**
toggle terminal
* code - `cmd + j`
* goland - `opt + fn + f12`

search: goland
* Search evrywhere - `shift + shift`
* Find any file or directly - `cmd + shift + o`
* Find a symbol - `cmd + shift + a`
* Find a type - `cmd + o`


Navigate backward
* vscode - `ctrl+-`
* goland - `cmd+[`

Navigate forward
* vscode - `shift+ctrl+-`
* goland - `cmd+]`

Find & select multiple occurences
* vscode - `cmd + d`
* goland - `ctrl + g`

Duplicate line or selection
* code - `opt+shift+up`
* goland - `cmd+d`

Move
* goland - `opt+shift+up`

Find 
* vscode 
* goland - `cmd+shift+f`


Find and replace
* vscode 
* goland - `cmd+shift+r`
